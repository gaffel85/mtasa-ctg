The game mode has a feature where a player that is far behind can press Z to catch up. The player is then teleported to a location closer to the leader. The position is based on the score of the player. If low score it will teleport close to the leader and if higher score it will end up somewhere between all players.

## The problem
Some inexperienced players have a hard time driving as fast as the other players and will always be left behind. I want to create a feature where those players can use a super catch up that will throw them into the middle of the action any time.

## The idea
The admin should be able to bring up a UI (with F3) where he can choose who will have the super catch up power. Multiple players can have it at the same time. The admin can toggle if they have the power or not. The window need a close button.
